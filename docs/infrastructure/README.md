# 基建文档入口

Status: Foundation Index.

本目录记录 StoryPlay 的横向基础设施能力。这里的内容不绑定单个页面，而是服务首页、游玩页、创作后台、SKU、模型调用、部署和计费等多个模块。

## 文档

- [Studio 存储 Roadmap](studio-storage-roadmap.md)：StoryProject、SKU 草稿、已发布 SKU、创作者素材的存储方向。
- [账号系统 Roadmap](account-system-roadmap.md)：账号、归属、权限、持久化、额度和用户数据路线图。

## 边界

基建负责：

- 账号和登录态。
- 权限和所有权校验。
- 用户维度的项目、故事、试玩、发布数据。
- 模型用量归因、额度、点数和账本。
- 数据库和媒体资源存储。
- 和部署强相关的运行时配置。

基建不直接负责：

- 创作后台具体编辑器 UI。
- 游玩页交互体验设计。
- 首页 SKU 卡片和推荐体验。
- 剧情生成质量和 agent 职责设计，除非涉及身份、额度或持久化。
